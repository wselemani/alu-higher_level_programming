The ``5-text_indentation`` module
===================================

Using ``text_indentation``
--------------------------

Importing the function:
    >>> text_indentation = __import__('5-text_indentation').text_indentation

Normal text parsing:
    >>> text_indentation("Hello. World? Testing: success")
    Hello.
    <BLANKLINE>
    World?
    <BLANKLINE>
    Testing:
    <BLANKLINE>
    success

Trailing spaces after characters are omitted:
    >>> text_indentation("First.    Second")
    First.
    <BLANKLINE>
    Second

Non-string type error validation:
    >>> text_indentation(12345)
    Traceback (most recent call last):
    TypeError: text must be a string

Missing argument error verification:
    >>> text_indentation()
    Traceback (most recent call last):
    TypeError: text_indentation() missing 1 required positional argument: 'text'
